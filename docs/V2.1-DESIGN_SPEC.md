## 0. 设计原则

V2.1 所有新增模块**严格沿用** V2.0 设计系统的以下基础规范，不做独立风格：

- **调色板**：`--bg-page: #e8ece4`, `--surface-default: #fafdf6`, `--surface-active: #e4ebe0`, `--text-primary: #1e2618`, `--text-secondary: #6b7862`, `--accent: #4f6d3a`, `--border-soft: rgba(30,38,24,0.08)`
- **字体**：Sans 主权（Inter/system-ui），Serif 仅用于和弦名/级数/调性选择器
- **圆角梯度**：面板 28px，卡片 20px，按钮胶囊 full，小卡片 16px
- **动效**：全部使用 Framer Motion spring 物理模型，拒绝线性过渡
- **容器**：`max-width: 440px`，安全区域适配 `env(safe-area-inset-*)`

本章节以下仅描述 V2.1 新增或变动的模块，未提及的部分沿用 V2.0 规范。

## 1. 播放栏改动

**视觉参数：**

- 文案：「节奏」

**状态：**

| 状态 | 背景 | 文字色 |
|------|------|--------|
| 默认（收起） | `bg-[#4f6d3a]/10` | `text-[#4f6d3a]` |

**动效：**


### 1.2 和弦时长滑块显隐

- 节奏型 OFF：和弦时长滑块正常显示（V2.0 行为不变）
- 节奏型 ON：和弦时长滑块隐藏，其空间被节奏面板展开区取代

```tsx

      {/* 和弦时长滑块 */}

```

### 3.2 单个开关按钮

**视觉参数：**

- 圆角：`rounded-full`
- 字号：`12px`，字重 `600`，字体 Sans
- 图标：lucide-react size=13，与文字间距 `gap: 6px`
- 布局：图标+文字水平居中

**三个按钮的图标映射：**

| 开关 | 图标 | 文案 |
|------|------|------|
| 节拍器 | `Metronome` | 节拍器 |
| 节奏型 | `Activity`（波形脉冲）| 节奏型 |
| 和弦 | `Music` | 和弦 |

**状态：**

| 状态 | 背景 | 文字色 | 阴影 |
|------|------|--------|------|
| ON（激活） | `bg-[#4f6d3a]` | `text-white` | `shadow-md shadow-[#4f6d3a]/25` |

| ON + Hover | `bg-[#3d552d]` | `text-white` | `shadow-md shadow-[#4f6d3a]/30` |

**动效：**
```tsx
// 开关按钮本体
<motion.button
  whileHover={{ scale: 1.03 }}
  whileTap={{ scale: 0.97 }}
  transition={{ type: "spring", stiffness: 400, damping: 16 }}
  className={`relative flex items-center justify-center gap-1.5 py-2.5 rounded-full
    text-xs font-semibold cursor-pointer select-none
    ${isOn ? 'bg-[#4f6d3a] text-white shadow-md' : 'bg-stone-200/50 text-[#6b7862]'}`}
>
  <Icon size={13} />
  <span>{label}</span>
</motion.button>
```

**无 layoutId 滑动指示器**：三开关是独立的 boolean 状态组合（不是互斥选项），因此不使用标签切换器的 `layoutId="pillow"` 动画。每个开关独立管理自己的选中态颜色渐变。

### 3.3 即时生效说明

开关切换立即生效，不等待播放启动。但音频行为在下一次播放启动时才体现：
- UI 状态即时更新（颜色、滑块显隐）
- 正在播放中切换：不中断当前播放，仅在下一个 tick/小节应用新模式

### 5.3 单个拍号按钮

**视觉参数：**
- 宽度：`min-w-[56px]`
- 高度：`py-2`
- 圆角：`rounded-full`
- 字号：`13px`，字重 `600`，字体 Sans（等宽数字）
- 文案：`4/4`、`3/4`、`6/8`

**选中态指示器（layoutId 滑动胶囊）：**

使用与 V2.0 创作面板标签切换器完全一致的 `layoutId` 技术：

```tsx
<div className="relative inline-flex gap-1 p-1 bg-stone-200/50 rounded-full border border-[#e4ebe0]/40">
  {['4/4', '3/4', '6/8'].map(ts => (
    <button
      key={ts}
      onClick={() => setTimeSignature(ts)}
      className="relative min-w-[56px] py-2 rounded-full text-[13px] font-semibold z-10 transition-colors"
      style={{ color: timeSignature === ts ? '#ffffff' : '#6b7862' }}
    >
      {timeSignature === ts && (
        <motion.div
          layoutId="ts_pillow"
          className="absolute inset-0 bg-[#4f6d3a] rounded-full shadow-sm"
          transition={{ type: "spring", stiffness: 380, damping: 22 }}
        />
      )}
      <span className="relative z-10">{ts}</span>
    </button>
  ))}
</div>
```

**注意**：`layoutId` 名称使用 `"ts_pillow"`，与创作面板标签的 `"main_action_pillow"` 区分，避免跨组件 layoutId 冲突导致动画错乱。

## 7. 预览网格

### 7.1 整体布局

网格位于节奏型选择器下方，是节奏型选择的视觉预览。

- 容器：`mt-3`，居中，带 `overflow-x: auto`（6/8 拍 24 格子时防止溢出）
- 拍间用竖线分隔，上方标注拍号数字

### 7.2 网格行结构

```
  1        2        3        4       ← 拍号数字
  ↓  ·  ·  ↓  ·  ↑  ↓  ·  ·  ↓  ·  ↑   ← 扫弦箭头
 ─── │ ─── │ ─── │ ─── │ ─── │ ───       ← 分隔线
```

### 7.3 拍号数字

- 字号：`9px`，字重 `600`，颜色 `text-[#6b7862]/70`
- 字体 Sans
- 位于每个拍子第一个格子的正上方
- 使用 `position: absolute` 或负 margin 定位

### 7.4 拍间竖线

- 使用 `border-left` 实现，位于每拍的第一个格子左侧
- 颜色：`border-[#e4ebe0]`，宽度 `1px`，高度撑满格子行
- 强拍（第 1 拍）分隔线：颜色 `border-[#4f6d3a]/30`，宽度 `1.5px`，微区分

### 7.5 单个格子

**视觉参数：**
- 宽度/高度：`24px × 28px`（桌面端），移动端可缩小至 `20px × 24px`
- 圆角：`4px`（`rounded-sm`）
- 背景：默认透明
- 文字：字号 `13px`，居中

**格子内容：**

| 标记 | 显示 | 含义 |
|------|------|------|
| `down` | `↓` | 低音→高音扫弦 |
| `up` | `↑` | 高音→低音扫弦 |
| `null` | `·` | 空拍（不触发新音） |

**空拍（`·`）颜色：** `text-[#d0d5cc]`（极淡灰绿，几乎不可见也不可忽略）
**扫弦箭头颜色：** `text-[#4f6d3a]`（↓↑ 均用强调绿色）
**强拍底色（第 1 拍所有格子）：** `bg-stone-100/60`（微弱的暖灰底区分）

### 7.6 播放高亮

当前播放位置的格子使用高亮背景 + 平滑过渡。

**高亮格：**
```tsx
className={`relative rounded-sm flex items-center justify-center
  ${isCurrentCell ? 'bg-[#4f6d3a]' : ''}
  ${isStrongBeat ? 'bg-stone-100/60' : ''}
`}
```

**高亮态视觉参数：**
- 背景：`bg-[#4f6d3a]`
- 箭头色：`text-white`
- 箭头微放大：`scale: 1.15`（用 motion.div 包裹）

**动效（格子高亮切换）：**
```tsx
<motion.span
  animate={{
    backgroundColor: isCurrentCell ? '#4f6d3a' : 'transparent',
    scale: isCurrentCell ? 1.05 : 1,
  }}
  transition={{ type: "spring", stiffness: 300, damping: 20 }}
  className="..."
>
  {slot === 'down' ? '↓' : slot === 'up' ? '↑' : '·'}
</motion.span>
```

**注意**：高亮格子在 tick 间切换（十六分音符级别，约 100-375ms），使用 spring 过渡确保视觉平滑不闪烁。

### 7.7 网格行完整渲染

```tsx
<div className="flex flex-col items-center gap-0.5 mt-3">
  {/* 拍号数字行 */}
  <div className="flex items-center" style={{ gap: 0 }}>
    {beats.map((beat, bi) => (
      <span
        key={bi}
        className="text-[9px] font-semibold text-[#6b7862]/70"
        style={{ width: beat.slots.length * 24 }}
      >
        {bi + 1}
      </span>
    ))}
  </div>

  {/* 格子行 */}
  <div className="flex items-center bg-stone-100/40 rounded-md p-0.5">
    {beats.map((beat, bi) => (
      <React.Fragment key={bi}>
        {/* 拍间竖线（非第一拍时显示） */}
        {bi > 0 && (
          <div className={`w-px h-7 ${bi === 0 ? 'bg-[#4f6d3a]/30' : 'bg-[#e4ebe0]'}`} />
        )}
        {beat.slots.map((slot, si) => {
          const isCurrent = currentBeat === bi && currentSlot === si;
          return (
            <motion.span
              key={si}
              animate={{
                backgroundColor: isCurrent ? '#4f6d3a' : (bi === 0 ? '#f0f3ed' : 'transparent'),
                scale: isCurrent ? 1.05 : 1,
              }}
              transition={{ type: "spring", stiffness: 300, damping: 20 }}
              className="w-6 h-7 flex items-center justify-center rounded-sm text-[13px] select-none"
              style={{ color: isCurrent ? '#ffffff' : slot === null ? '#d0d5cc' : '#4f6d3a' }}
            >
              {slot === 'down' ? '↓' : slot === 'up' ? '↑' : '·'}
            </motion.span>
          );
        })}
      </React.Fragment>
    ))}
  </div>
</div>
```

## 8. 和弦卡片播放高亮（增强）

V2.0 已有 `activeChordIndex` 高亮逻辑（激活态：`bg-[#e4ebe0]` + `ring-2 ring-[#4f6d3a]/20`）。

V2.1 增强：在节奏型模式下，当前和弦卡片增加**呼吸动效**，让卡片在播放时有轻微的「活着的」感觉。

```tsx
// 当前和弦卡片（节奏型模式下）
animate={{
  scale: isActive && isRhythmMode ? [1, 1.008, 1] : isActive ? 1.015 : 1,
  backgroundColor: isActive ? '#e4ebe0' : '#fafdf6',
  boxShadow: isActive
    ? '0 4px 12px rgba(30,38,24,0.06), 0 0 0 2px rgba(79,109,58,0.15)'
    : '0 1px 3px rgba(30,38,24,0.02)',
}}
transition={{
  scale: { repeat: isActive && isRhythmMode ? Infinity : 0, duration: 0.6, ease: "easeInOut" },
  backgroundColor: { type: "spring", stiffness: 400, damping: 16 },
}}
```

**注意**：呼吸动效仅在节奏型 ON + 正在播放时生效。V2.0 的匀速琶音模式保持原有行为。

```tsx
<motion.span
```

脉冲时机对齐强拍（第 1 拍），使用 `useEffect` + BPM 计算间隔。

## 11. 响应式适配

### 11.1 容器宽度

所有新增组件在 `max-width: 440px` 容器内，与 V2.0 一致。

### 11.2 网格溢出处理

6/8 拍 × 4 个十六分音符 = 24 格 × 24px = 576px，超出 440px 容器宽度。

**处理：**
- 网格容器 `overflow-x: auto`，允许水平滚动
- 使用 `scrollbar-hide`（隐藏滚动条但可滑动）
- 默认滚动到高亮格子可见（`scrollIntoView` 或手动计算 `scrollLeft`）
- 移动端格子缩小至 `20px`，24×20 = 480px，仍略超容器，水平滚动为主要适配策略

### 11.3 节奏面板间距适配

移动端（≤440px）面板内边距从 `p-4` 缩减至 `p-3.5`，行间距从 `14px` 缩减至 `12px`。
