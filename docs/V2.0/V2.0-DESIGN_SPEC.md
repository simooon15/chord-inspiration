# 和弦灵感生成器 — 视觉设计系统 (V2.0-DESIGN_SPEC)

> 森林质感莫兰迪、严格中西文双轨排版与物理弹性阻尼运动统一规范
> 面向对象：UI/UX 设计师、重构工程师、AI 编码 Agent
> 设计信仰：抵抗同质化的工业大厂高饱和冷光扁平风格，倡导温润、低对比、富有呼吸感的人文本源主义

---

## 1. 设计哲学

### 1.1 去工业化

拒绝死黑 (`#000000`) 和刺眼白 (`#ffffff`)。所有颜色自带泥土、草木与老纸张的灰度。

### 1.2 材质分级

通过圆角梯级、纸张重叠阴影和阻尼动效，向用户传递物理现实中「棉宣、树皮、树荫」的叠加反馈。

### 1.3 中文非衬线主权

除标题「和弦灵感」、和弦名、级数、调性选择器外，所有界面汉字锁死无衬线体，规避宋体在小字号下产生的毛刺与断线。

## 3. 排版系统

### 3.1 字体矩阵

| 用途 | 字体链 | 定义位置 |
|------|--------|---------|
| **Serif** — 和弦名、级数、标题「和弦灵感」、调性选择器 | `'Iowan Old Style', 'Palatino', 'Noto Serif SC', 'Songti SC', serif` | ChordCard.tsx / App.tsx 行内 style |
| **Sans** — 所有 UI 文字（按钮、标签、说明、弹窗、输入框） | `'Inter', system-ui, -apple-system, sans-serif` | index.css `--font-sans` |
| **Mono** — TAB 指法谱、技术标注、计时器、录制列表数据 | `'JetBrains Mono', SFMono-Regular, monospace` | index.css `--font-mono` |

### 3.2 Serif 适用范围

Serif 仅限于以下极少数核心元素，保留老和弦谱书的古朴质感：

- 页面标题「和弦灵感」
- 和弦名（Cmaj7、Am、F△7）
- 级数标记（I、vi、IV、V、♭VII）
- 调性选择器选项（C 调、Db 调）

### 3.3 Sans 主权说明

中文宋体（衬线）在 12-14px 的小字号下，横线极细易断，在低亮度屏幕上产生视觉疲劳。因此，12px 以下的所有中文字符**禁止使用衬线体**，锁死无衬线系统。

| 元素 | 字号 | 字重 | 字体 |

## 6. SVG 吉他弦图坐标系统

和弦卡片内嵌离线高保真 SVG 弦图，不加载任何外部图片资源。

### 6.1 坐标系

```
     0   1   2   3   4   品 （Y-offset: 16px/品）
    ========================  Y = 24  （弦枕，baseFret===1 时加粗）
6→1弦: |   |   |   |   |
X = 24+i*12              |   |   |   |   |    j=1 → Y = 40
    ------------------------
    |   O   |   |   |   |    按指高亮点 r=4.2px  fill="#4f6d3a"
    ------------------------  j=3 → Y = 72
    |   |   |   |   |
    ------------------------  j=4 → Y = 88
```

### 6.2 SVG 参数

| 参数 | 值 |
|------|-----|
| 视口 | `viewBox="0 0 100 110"` |
| SVG 容器尺寸 | 98 × 106px |
| 弦纵向起始 X | 24 + i × 12（i = 0-5）|
| 品横向起始 Y | 24 + j × 16（j = 0-4）|
| 低音 E 弦粗 | 1.6px，其余弦 0.9px |
| 弦枕线粗（baseFret===1） | 3px |
| 按指点半径 | 4.2px |
| 按指点颜色 | `#4f6d3a` fill，`#fafdf6` stroke 0.8px |
| 开放弦标记 O | 半径 2.5px，空线圆 |
| 闷音标记 X | `#dc2626` 双线叉，透明度 0.65 |
| 基准品号文字 | 字号 9.5px，fill `#4f6d3a` |

### 6.3 横按偏移算法

1. 取按品最小值 `FretMin`，最大值 `FretMax`
2. 若 `FretMax > 4` 且 `FretMin > 1`，触发偏移
3. `baseFret = FretMin`，所有按品位偏移 `- baseFret`
4. 弦图左侧标注 `{baseFret}f`（如 `5f`、`8f`）
5. 实现于 [guitarChords.ts](../src/guitarChords.ts)
