# AI Coding Agent Guidelines

- **Minimalist Aesthetic (No Descriptive Fluff)**: Keep the page layout extremely clean, elegant, and professional. Avoid adding wordy, hand-holding, instructional or explanatory sentences. Labels and placeholders must remain humble and literal.
- **Framer Motion Springs**: Ensure interactive states and sliders translate with organic, elastic, bouncy springs (`type: "spring"`) for premium material quality.
- **Typography & Font Choices (Chinese Font Rule)**: Except for the main title "和弦灵感" (Chord Inspiration) and the key-selector dropdown (选调子的模块) where serif fonts (Noto Serif SC / Songti SC) can be used, all other Chinese characters on the interface MUST strictly use standard sans-serif (非衬线字体, e.g. system-ui sans-serif / 'Inter') to protect aesthetic integrity.
